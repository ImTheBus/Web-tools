About this site
hover to expand

IMTHEBUS is a small, evolving collection of tools built for practical use in day to day work. The aim is to make it easier to run better conversations and make clearer decisions without needing a full platform or another login.

Each tool is designed to be:

  • Lightweight – open it in a browser and you are ready.
  • Focused – each tool solves one problem instead of trying to do everything.
  • Conversation first – intended to be used with other people, not just alone.
Whiteboard tips
shared across tools

This home page shows your tools as notes on the shared board. Inside each tool you will see the same whiteboard frame, with keyboard and mouse patterns for working with notes.

Common patterns include 1 2 3 4 for modes or quadrants, enter to add a note, drag to move notes, and double click to edit.
